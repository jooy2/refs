# 생산성 업무

- **[Trello](https://trello.com)** `영`

  - 프로젝트 작업 관리, 진행 상황을 한 눈에 보거나 TO-DO 리스트 작성을 위한 웹 애플리케이션

- **[Todoist](https://todoist.com)** `영`

  - 할 일 목록을 작업 단위로 관리할 수 있는 웹 애플리케이션

- **[TeamGantt](https://www.teamgantt.com)** `영`

  - 프로젝트 스케줄 관리 웹 애플리케이션
